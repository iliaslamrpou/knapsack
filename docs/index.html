Knapsack Solver

What is Knapsack solver

Knapsack solver is a gitHub repository that contains tools and C++ code about the knapsack 0-1 problem
The generator folder contains two very helpful tools to create one knapsack 0-1 problem or a pack of problems
The problems can solved using 6 algorithms

Algorithms:

  1. Greedy approach
  2. Brute force
  3. Branch and bound
  4. Dynamic programming
  5. Dynamic OR-Tools solver
  6. Integer OR-Tools solver

The knapsack.exe file is a powerful tool that can create and solve knapsack problems and extracts the results to files so as to be easy the statistical analysis.

More info

             

Using the knapsack tools that included on the knapsack repository you we created  320 knapsack problems and then we solved them using all algorithms

The results were stored on a CSV file.

The two Dynamic programing algorithms and the Integer programing algorithm can give the best solution for all the problems and they need very little time. 

The Greedy approach algorithm can't calculate the best solution (Blue line)

The Brute force algorithm (Red line) needs too much time to solve problems with 50 items and above. The 10 seconds time out limit stopped the algorithm to find the best solution

The Branch and bound algorithm seems a litle better then the Brute force. For many problems it needs more than 10 seconds to return a solution
